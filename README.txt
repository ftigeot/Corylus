This is Corylus, a web-browser based ERP project

It can manage products, customer and supplier relationships, create
quotations, invoices, delivery slips, order forms, etc...

It is mostly written in Ruby on Rails with a few PostgreSQL stored
procedures to improve performance.

Professional-looking PDF documents can be created instantly with the
help of XSL-FO stylesheets and Java servlets.

Documents can also be produced in XML formats, suitable for automation
with external tools.
